# 웹 관리 기능 메모

## 현재 제공 기능
- 거래 검색
- 거래 유형/계정/카테고리 필터
- 기간 프리셋 + 직접 날짜 범위 조회
- 필터 결과 CSV / XLSX export
- 필터 링크 복사(공유/복원용)
- 사용자 기본 설정 저장
  - 표시 이름
  - 기본 통화
  - 주 시작 요일
  - 월 시작일
- 관리 목록 CRUD 보강
  - 계정/카테고리: 보관(soft-archive)
  - 예산/반복거래: 삭제
- 대시보드 통계
  - 이번 달 요약(수입/지출/잔액)
  - 카테고리 소비 비중
  - 예산 진행률
  - 최근 6개월 추이 차트
- 반복배치 실행 로그
  - 최근 7일 성공/실패 건수
  - 총 누적 성공/실패 건수
  - 마지막 실행 시각
  - 최근 7일 실패 사유 요약(상위 사유 집계)
  - 실패 로그 상세(최근 5건 우선 + 전체 펼쳐보기)
  - 실패 사유 검색 필터
  - 실패 로그 기간 필터(최근 7일/30일/전체)
  - 실패 로그 사유 상태 필터(사유 있음/없음)
  - 실패 로그 필터 상태 로컬 저장(재접속 시 유지)
  - 실패 로그 필터 URL 동기화(공유/새로고침 복원)
  - 반복 로그 필터 URL 하네스 검증: `pnpm ops:harness:web`

## 모바일/웹 역할 분리
- 모바일: 빠른 입력, 한 손 사용성, 당일 기록 중심
- 웹: 검토/분석/관리 중심(필터, 내보내기, 설정)

## 내보내기 정책
- CSV는 UTF-8 BOM 기준으로 저장
- XLSX도 동일 필터 결과를 기준으로 생성
- 추후 보고서 템플릿 확장 시 현재 필터 데이터 파이프라인 재사용 가능

## 다음 개선 후보
- 다중 선택 필터
- 정렬 옵션(금액/일자/카테고리)
