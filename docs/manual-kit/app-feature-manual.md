# 앱 기능 매뉴얼 (현재 버전)

기준일: 2026-04-13
대상: 가계부2 MVP (모바일 + 웹)

## 1) 앱 개요
- 모바일과 웹이 같은 Supabase 계정/DB를 공유합니다.
- 초기 MVP는 수동 입력 중심이며, 거래 타입은 수입/지출/이체를 지원합니다.
- 반복거래는 등록/수정 + 배치 실행 로그 확인 흐름까지 포함합니다.

## 2) 핵심 기능 목록

### 공통 도메인
- 사용자/프로필
- 계정(Account)
- 카테고리(Category)
- 거래(Transaction)
- 예산(Budget)
- 반복거래(Recurring Transaction)

### 거래 타입
- income(수입)
- expense(지출)
- transfer(이체)

## 3) 모바일 기능과 사용법

### 홈
- 이번 달 수입/지출/잔액 요약
- 최근 거래 내역
- 예산 요약
- 반복지출 요약
- 반복거래 실행 로그 요약
: 최근 7일 성공/실패, 총 누적 성공/실패, 최근 로그 카드

### 입력
1. 거래 타입 선택(수입/지출/이체)
2. 금액 입력(빠른 금액 버튼 사용 가능)
3. 날짜 입력(오늘 버튼 지원)
4. 계정 선택
5. 카테고리(또는 이체 대상 계정) 선택
6. 메모 입력 후 저장

### 목록
- 거래 목록 조회
- 항목 수정
- 항목 삭제(soft delete 흐름 기준)

### 관리
- 계정 등록/수정
- 카테고리 등록/수정
- 예산 등록/수정
- 반복거래 등록/수정

## 4) 웹 기능과 사용법

### 대시보드/분석
- 이번 달 수입/지출/잔액 카드
- 카테고리 소비 비중
- 예산 진행률
- 최근 6개월 추이

### 거래 검토
- 검색(메모/계정/카테고리)
- 필터(유형/계정/카테고리/기간)
- 필터 링크 복사
- CSV / XLSX 내보내기

### 반복배치 실행 로그
- 최근 7일 성공/실패
- 총 누적 성공/실패
- 마지막 실행 시각
- 실패 사유 요약(상위 사유)
- 실패 로그 상세(최근 5건 우선 + 전체 펼쳐보기)

### 설정
- 표시 이름
- 기본 통화
- 주 시작 요일
- 월 시작일

## 5) 반복거래 배치 운영
- Edge Function: run-recurring-batch
- 검증 명령: pnpm ops:check-recurring-batch
- 관련 문서: docs/recurring-batch-scheduler.md

## 6) 빠른 점검 체크리스트
1. 모바일/웹 동일 계정 로그인 확인
2. 모바일 거래 입력 후 웹 반영 확인
3. 웹 수정 후 모바일 반영 확인
4. 웹 필터 + 내보내기(CSV/XLSX) 확인
5. 반복배치 dry-run 확인
